# GYM DIARY BACKEND

Treniņu dienasgrāmatas lietotnes backend. Projekts izstrādāts skolas pētnieciskā darba ietvaros (11. klase). Tajā ir īstenots REST API lietotāju autorizācijai un treniņu pārvaldībai.

## Tehnoloģijas

- **Java 21**
- **Spring Boot 3.1.0**
- **Spring Security** + **JWT** (autentifikācija ar žetoniem)
- **Maven** (kompilācija)
- **ConcurrentHashMap** (datu glabāšana atmiņā, bez datu bāzes)

##  Palaišana

### Prasības
- JDK 21
- Maven (vai izmantojiet iebūvēto `./mvnw`)

### Soļi
```bash
# klonēt repozitoriju
git clone https://github.com/your-username/gym-diary-backend.git
cd gym-diary-backend

# kompilēt un palaist
